# IELTS Practice Tracker

Checklist harian untuk latihan IELTS (Cambridge Book 1-17 + ieltstrainingonline.com)
menuju 30 September 2026, target band 8.5-9.

Lihat tutorial deploy lengkap di chat Claude, ringkasannya:

1. `npm install`
2. Push ke GitHub
3. Import ke Vercel
4. Tambah Vercel KV (Storage tab) -> otomatis isi env var
5. Set env var `APP_PIN` (PIN bebas, buat kamu doang yang tau)
6. Deploy -> buka link-nya dari HP & laptop, datanya sama
